2 How to install and start ZenZefi

This chapter describes the installation of ZenZefi on a local machine.

!

  • ZenZefi can be upgraded by installing a newer version using the ZenZefi Installer.

  • ZenZefi upgrade with huge leaps in versions it is not advised.

 

ZenZefi is provided as an MSI-Installer for Windows systems (32bit and 64bit versions) and as a runnable Java Jar-file for Linux systems. Both variants can be downloaded from ZenZefi's homepage. ZenZefi's MSI-installer installs ZenZefi as a Windows service, so that ZenZefi automatically starts after a successful installation and on all subsequent Windows restarts. See chapter 2.1.1 ZenZefi installation as a Windows service.

2.1 How to install ZenZefi via MSI-installer

 

i

During the ZenZefi installation, the user is notified if  the wrong .NET version is installed on the target Windows system. In case of a missing .NET version the installation process stops.

 

ZenZefi MSI-Windows installer checks if a another version of Zenzefi is already installed on the system. If no ZenZefi version is found, the normal installations process starts (see chapter 2.1.1 ZenZefi installation as a Windows service). If a previous ZenZefi version is found, ZenZefi Windows installer asks the user how to continue with the installation process. Possible options:

  • if the existing ZenZefi version on the system is older than the one the user wants to install, a update can be performed. Press the "Update" button:

  

  Figure 2: ZenZefi Update

  • if the existing ZenZefi version on the system is newer than the one the user wants to install, a rollback can be performed. Press the "Rollback" button:

  

  Figure 3: ZenZefi Rollback

  • in both of the cases presented above, the user can press the "Clean Installation" button. This action uninstalls the ZenZefi version already existing on the system and installs the ZenZefi version desired by the user. The following directories are removed: C:\ProgramData\ZenZefi and C:\Program Files\ZenZefi

The Silent Installer does not ask the user if ZenZefi should be updated (or downgraded). It continues the installation as normal. See chapter 2.1.2 ZenZefi Silent Installer.

2.1.1 ZenZefi installation as a Windows service

1.  Execute ZenZefi.msi

  

  Figure 4: ZenZefi Installation As A Windows Service

 

2.  After the ZenZefi Installer window appears, press the "Next" button.

  

  Figure 5: ZenZefi Installer - Startup Page

 

3.  Press the "Install" button to begin the ZenZefi installation.

  

  Figure 6: ZenZefi Installer - Ready To Install

 

ZenZefi is installed into default path "C:\Program Files\ZenZefi". The progress bar will be visible until the installation is complete.

  

    Figure 7: ZenZefi Installer - In Progress

 

4.  Press the "Finish" button to close the ZenZefi Installer. The window will disappear

  

  Figure 8: ZenZefi Installer - Finish Installation

 

In Windows Task Manager Services tab, the ZenZefi process exists and is running.

The ZenZefi program also exists in Control panel Programs Programs and Features.

  

      Figure 9: ZenZefi In Windows Task Manager And Control Panel

 

The ZenZefi Windows start menu folder contains:

  • Uninstall ZenZefi: starts the ZenZefi uninstall process

  • User Manual: opens the ZenZefi user manual in PDF format

  • ZenZefi Rest-API: opens the Swagger in the default browser

  • ZenZefi-UI: opens ZenZefi UI in the default browser

  

  Figure 10: ZenZefi Program In Windows Start Menu

2.1.2 ZenZefi Silent Installer

ZenZefi can also be installed in silent mode, i.e. no user interactions are needed. This is done as follows:

1.  Open cmd.exe as Administrator

2.  Run the command: msiexec.exe /i ZenZefi.msi /qn

  

  Figure 11: ZenZefi Silent Installer

!

The Silent Installer can be run with configured logging path: msiexec.exe /i ZenZefi.msi /qn /L*V "<PATH>\installer.log. The <PATH> folder, i.e. C:\ProgramData\Zenzefi must already exist.

2.2 How to uninstall ZenZefi

2.2.1 Uninstall ZenZefi as a Windows service

1.  Execute "Uninstall ZenZefi" from the Windows Start Menu

  

  Figure 12: ZenZefi Uninstall - Windows Start Menu

 

2.  Confirm the uninstallation process, by pressing "Yes".

  

  Figure 13: ZenZefi Uninstall - Dialog

 

Uninstallation deletes all files created during installation and during operation of ZenZefi. Moreover, the uninstaller also removes all ZenZefi entries from Windows service settings. The progress bar will be visible until the uninstallation is complete.

  

  Figure 14: ZenZefi Uninstall - in Progress

2.2.2 Silent uninstall

1.  Open cmd.exe as Administrator

2.  Run the command: msiexec.exe /i /qn ZenZefi.msi

2.3 How to stop ZenZefi

ZenZefi provides a command to shut down ZenZefi via the CLI.

Invoke ZenZefi CLI with the command line parameter "-shutdown", ZenZefi is shut down and the process exits completely. Complete command is: zenzefi_cli.exe -shutdown
